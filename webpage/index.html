Jira4Android - Mobile Task Management

MOBILE TASK MANAGEMENT

Have you ever imagine that you can manage task and issues everywhere you want? Now it's possible! With Jira4Android application, you can do it without hindrance using your mobile phone.


Features

Filters

Task details

Comments

Search issues that interests you through using filters - both local and remote (from server).

When you find suitable issue, you can see its details. Jira4Android gives information about assignees, reporters, logged work, resolution and much more.

Do you want to write something about any issue when you don't have an access to a computer? No problem! Now you can comment anything on Jira everywhere you want.
