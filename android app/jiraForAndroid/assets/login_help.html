Login screen
This screen let you connect to Jira server.

In the first field, named "http://example.org", you should enter a jira server address you would like to connect.

In the second field, named "Username", you should enter your username.

In the third field, named "Password", you should enter your password.

Checking option "Remember me" makes, that application will remember your data and log you in automatically during start.

If you entered correct data press login button. It will run project list screen.

If data you entered was incorrect, you will get a message what to change.

When you want to log out, open menu and choose logout.
