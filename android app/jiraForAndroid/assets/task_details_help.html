Issue details
This screen contains details about selected issue.

The left tab, "comments", let you see list of comments posted to this issue. You can also add new comment by entering your text and clicking "Send".

The center tab, "Basic Info", shows you the main information about selected issue.

The right tab, "Work Log", let you see the work log of this issue.

You can refresh this screen by clicking on the arrow icon in the right top corner.

You can return to the project list by clicking on the home icon in the left top corner.
