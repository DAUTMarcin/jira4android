Filters list screen
This screen has two tabs with filters.

Left tab, "Local", contains list of filters defined by default in the application.

Right tab, "Server Filters", contains list of filters defined by you on the jira server.

You can see issues on server by simply clicking on the selected filter.

You can refresh the filters list by clicking on the arrow icon in the right top corner.

You can return to the project list by clicking on the home icon in the left top corner.
